미니게임

짧은 휴식 시간, 픽셀 감성으로 즐겨보세요

게임 선택

🔢 숫자 맞추기
  • 1부터 100 사이 숫자 추리 게임
  • 입력할 때마다 Up/Down 힌트 제공
  • 시도 횟수 표시와 즉시 리셋 지원
🐍 스네이크
  • 키보드/터치로 간단 조작
  • 먹이를 먹을수록 길이 증가
  • 최고 점수 기록 저장
❎ 틱택토
  • 한 판이 짧아 가볍게 즐기기 좋음
  • 선공 전환과 즉시 재시작 지원
  • PC/모바일 모두 클릭으로 플레이
✊ 가위바위보
  • 클릭 한 번으로 즉시 한 판
  • 사용자 vs 컴퓨터 점수판 제공
  • 리셋 버튼으로 점수 초기화
🪨 님 게임
  • 돌을 1~3개씩 가져가는 턴제 전략 게임
  • 마지막 돌을 가져가면 승리
  • 짧은 판으로 두뇌 워밍업에 적합
🏓 퐁
  • 패들로 공을 받아치는 클래식 아케이드
  • 키보드와 모바일 버튼 모두 지원
  • 시작/일시정지와 즉시 리셋 가능
🧩 행맨
  • 알파벳 한 글자씩 단어를 맞추는 퍼즐
  • 오답 6번 전까지 정답을 완성해야 승리
  • 새 단어 버튼으로 바로 다음 판 시작
🃏 메모리 매치
  • 카드를 뒤집어 같은 짝을 찾는 기억력 게임
  • 총 이동 횟수로 기록을 겨뤄볼 수 있음
  • 짧은 시간 집중력 훈련에 적합
🔴 커넥트4
  • 7x6 보드에 디스크를 떨어뜨리는 전략 게임
  • 가로·세로·대각선 4개를 먼저 만들면 승리
  • 2인 턴제로 빠르게 한 판 가능
💡 라이츠아웃
  • 타일과 인접 타일이 함께 토글되는 퍼즐
  • 모든 불을 끄면 클리어
  • 최소 이동 수를 노리는 최적화 플레이 가능
💣 지뢰찾기
  • 안전한 칸을 모두 열면 승리하는 퍼즐
  • 우클릭(모바일 길게 터치 대체)으로 깃발 표시
  • 숫자 힌트로 지뢰 위치를 추론
🎵 사이먼
  • 빛나는 순서를 기억해 따라 누르는 게임
  • 라운드가 올라갈수록 시퀀스 길이 증가
  • 기억력과 반응 속도 훈련에 적합
🔲 2048
  • 같은 숫자 타일을 합쳐 2048에 도전
  • 방향키로 간단 조작
  • 짧은 시간 두뇌 회전에 적합
🂡 블랙잭
  • 딜러와 21에 가까운 점수 대결
  • Hit/Stand 기본 규칙으로 빠르게 플레이
  • 운과 판단을 함께 쓰는 카드 게임
🧱 브레이크아웃
  • 패들로 공을 튕겨 벽돌을 모두 파괴
  • 키보드와 마우스 모두 지원
  • 아케이드 감성의 클래식 게임
🗼 하노이 탑
  • 작은 원판을 활용해 전체 스택 이동
  • 큰 원판 위에 작은 원판만 올릴 수 있음
  • 논리적 사고력 훈련에 효과적
🎲 피그 다이스
  • 주사위를 굴려 점수를 쌓는 턴제 게임
  • 1이 나오면 해당 턴 점수는 0점
  • 언제 멈출지 결정하는 리스크 관리 재미
