짧게 쉬어갈 때도 완성도 있게

AgeCalc의 미니게임은 사이트 전체 브랜드를 유지하되, 더 대담한 컬러와 조금 더 빠른 리듬으로 설계한 플레이 공간입니다.

Quick 짧은 한 판 중심
Touch 모바일 조작 대응
Arcade 가볍고 선명한 플레이 감각

게임 선택

전략, 퍼즐, 반응 속도, 추리처럼 다른 리듬의 게임을 한곳에 모았습니다.

🔢 숫자 맞추기
  • 1부터 100 사이 숫자 추리 게임
  • 입력할 때마다 Up/Down 힌트 제공
  • 시도 횟수 표시와 즉시 리셋 지원
🐍 스네이크
  • 키보드/터치로 간단 조작
  • 먹이를 먹을수록 길이 증가
  • 최고 점수 기록 저장
❎ 틱택토
  • 한 판이 짧아 가볍게 즐기기 좋음
  • 선공 전환과 즉시 재시작 지원
  • PC/모바일 모두 클릭으로 플레이
✊ 가위바위보
  • 클릭 한 번으로 즉시 한 판
  • 사용자 vs 컴퓨터 점수판 제공
  • 리셋 버튼으로 점수 초기화
🪨 님 게임
  • 돌을 1~3개씩 가져가는 턴제 전략 게임
  • 마지막 돌을 가져가면 승리
  • 짧은 판으로 두뇌 워밍업에 적합
🏓 퐁
  • 패들로 공을 받아치는 클래식 아케이드
  • 키보드와 모바일 버튼 모두 지원
  • 시작/일시정지와 즉시 리셋 가능
🧩 행맨
  • 한글 글자를 하나씩 맞춰 단어를 완성하는 퍼즐
  • 오답 6번 전까지 정답을 완성해야 승리
  • 새 단어 버튼으로 바로 다음 판 시작
🃏 메모리 매치
  • 카드를 뒤집어 같은 짝을 찾는 기억력 게임
  • 총 이동 횟수로 기록을 겨뤄볼 수 있음
  • 짧은 시간 집중력 훈련에 적합
🔴 커넥트4
  • 7x6 보드에 디스크를 떨어뜨리는 전략 게임
  • 가로·세로·대각선 4개를 먼저 만들면 승리
  • 2인 턴제로 빠르게 한 판 가능
💡 라이츠아웃
  • 타일과 인접 타일이 함께 토글되는 퍼즐
  • 모든 불을 끄면 클리어
  • 최소 이동 수를 노리는 최적화 플레이 가능
💣 지뢰찾기
  • 안전한 칸을 모두 열면 승리하는 퍼즐
  • 우클릭(모바일 길게 터치 대체)으로 깃발 표시
  • 숫자 힌트로 지뢰 위치를 추론
🎵 사이먼
  • 빛나는 순서를 기억해 따라 누르는 게임
  • 라운드가 올라갈수록 시퀀스 길이 증가
  • 기억력과 반응 속도 훈련에 적합
🔲 2048
  • 같은 숫자 타일을 합쳐 2048에 도전
  • 방향키로 간단 조작
  • 짧은 시간 두뇌 회전에 적합
🂡 블랙잭
  • 딜러와 21에 가까운 점수 대결
  • 카드 받기와 멈추기 규칙으로 빠르게 플레이
  • 운과 판단을 함께 쓰는 카드 게임
🧱 브레이크아웃
  • 패들로 공을 튕겨 벽돌을 모두 파괴
  • 키보드와 마우스 모두 지원
  • 아케이드 감성의 클래식 게임
🗼 하노이 탑
  • 작은 원판을 활용해 전체 스택 이동
  • 큰 원판 위에 작은 원판만 올릴 수 있음
  • 논리적 사고력 훈련에 효과적
🎲 피그 다이스
  • 주사위를 굴려 점수를 쌓는 턴제 게임
  • 1이 나오면 해당 턴 점수는 0점
  • 언제 멈출지 결정하는 리스크 관리 재미
⚫ 오목
  • 15x15 보드에서 돌 다섯 개를 먼저 잇는 게임
  • 흑과 백이 번갈아 두는 클래식 전략 보드
  • 짧은 규칙으로 깊은 수읽기를 즐길 수 있음
◐ 리버시
  • 상대 돌을 뒤집어 더 많은 칸을 차지하는 게임
  • 유효한 수만 둘 수 있어 전개 판단이 중요
  • 후반으로 갈수록 역전 가능성이 큰 전략 게임
✏️ 점 잇기
  • 선을 그어 상자를 완성하면 점수를 얻는 게임
  • 상자를 만든 플레이어는 한 번 더 둘 수 있음
  • 끝수 읽기가 중요한 고전 퍼즐 보드게임
🪺 만칼라
  • 돌을 뿌리며 저장소 점수를 쌓는 전략 게임
  • 상대 돌을 가져오는 타이밍이 핵심
  • 한 턴 더 얻는 수읽기가 중요한 보드게임
🎯 마스터마인드
  • 숨겨진 4색 조합을 추리하는 코드 브레이킹 게임
  • 정확한 위치와 색상 힌트로 정답을 좁혀감
  • 논리 추론과 경우의 수 계산이 핵심
🃏 카드 전쟁
  • 서로 한 장씩 뒤집어 높은 카드가 승리
  • 동점이면 전쟁이 이어지는 단순 규칙 카드게임
  • 짧은 템포로 가볍게 즐기기 좋음
🚢 해전
  • 5x5 보드에 숨은 함선을 추리해 찾아내는 게임
  • 명중과 빗나감 정보를 바탕으로 다음 칸을 읽어야 함
  • 짧은 턴 안에 전체 함선을 침몰시키는 것이 목표
🔴 체커
  • 대각선 이동과 점프로 상대 말을 잡는 보드게임
  • 잡을 수 있는 말이 있다면 반드시 점프해야 함
  • 왕 승급 이후 이동 폭이 넓어지는 전략 게임
1️⃣ 15 퍼즐
  • 빈칸을 이용해 숫자 타일을 순서대로 맞추는 퍼즐
  • 한 칸씩 밀어가며 최소 이동을 노릴 수 있음
  • 짧은 시간 집중력 훈련에 적합
📍 페그 솔리테어
  • 핀을 하나씩 점프해 제거하는 클래식 퍼즐
  • 마지막에 핀 하나만 남기면 클리어
  • 끝수 계산과 경로 계획이 중요한 게임
🎲 야추
  • 주사위를 굴린 뒤 가장 유리한 점수 칸을 선택
  • 고정 기능으로 원하는 조합을 노릴 수 있음
  • 13라운드 합산으로 총점을 겨루는 게임
